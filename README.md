# django-nextjs

- django as a backend
- nextjs as a frontend

## to run:
### the backend:
- this project uses uv because its literally the goat (google python uv)
- `cd backend`
- `uv run manage.py runserver`

### the frontend:
- `cd frontend`
- `npm i`
- `npm run dev`

### viewing:
- `localhost:8000` for the backend
- `localhost:3000` for the frontend
